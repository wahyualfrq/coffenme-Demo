# ☕ Coffee & Me - Official Website

Welcome to the official repository for **Coffee & Me**, a premium coffee shop located in Palembang, Indonesia. This website is designed to provide an elegant, cozy, and user-friendly experience for our customers, helping them explore our menu, view our gallery, and easily make reservations or orders via WhatsApp.

## 🌟 Key Features

- **Modern & Elegant Design**: Clean aesthetics that reflect the cozy and premium atmosphere of our coffee shop.
- **Responsive Layout**: Fully optimized for mobile, tablet, and desktop viewing.
- **Interactive Menu**: Explore our signature main courses and refreshing beverages.
- **Direct PDF Menu Download**: Easy access to our complete menu in PDF format.
- **WhatsApp Integration**: Seamless connection for direct orders and reservations.
- **Dynamic Gallery**: Visual showcase of our comfortable spaces, from the main dining area to our co-working spaces.
- **Customer Reviews**: Authentic feedback from our happy patrons.

## 🚀 Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Language**: TypeScript

## 📦 Getting Started

First, clone the repository and install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📁 Project Structure

- `/src/app` - Next.js app router pages and layouts.
- `/src/components` - Reusable UI components (Navbar, Hero, Menu, Gallery, etc.).
- `/src/data` - Static data files (business info, menu items).
- `/public` - Static assets like images, logos, and the PDF menu.
